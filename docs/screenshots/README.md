## 截图要求

1. **尺寸**: 建议宽度 800-1200px，保持比例清晰
2. **格式**: PNG 格式（保证文字清晰）
3. **内容**: 截图应包含完整的页面布局，必要时标注关键功能区域
4. **时机**: 在功能正常运行时截图，避免截取错误状态

---

## 截图工具推荐

- **浏览器**: Chrome DevTools 截图功能 (Ctrl+Shift+P → "Capture screenshot")
- **系统**: macOS 截图 (Cmd+Shift+4)，Windows 截图工具
- **标注**: Snipaste、PixPin 等标注工具

---

## 截图检查流程

1. 按清单顺序依次截图
2. 检查截图清晰度和完整性
3. 重命名文件为规范命名
4. 放入本目录
5. 在报告中引用对应截图
